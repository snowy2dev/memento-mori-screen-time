# Memento Mori — Screen Time Reflection Tool

A minimal web app that calculates how much of your life has been spent on screens.

Instead of daily limits or productivity advice, this tool presents a single number — your total screen time — to encourage reflection.

## Live Demo
https://screentime-mori.netlify.app/

## Concept

Most screen time apps focus on daily usage.

This project focuses on lifetime accumulation.

Seeing your usage in years instead of hours changes how it feels.

## Features

- Lifetime screen time calculation based on user input
- Animated count-up for emotional impact
- Percentage of waking life spent on screens
- Interactive projection of future usage
- Mobile-first design styled as a virtual phone interface

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Design Philosophy

- No judgment, no advice
- Minimal interface
- Let the user interpret the result
